cleaning up the pedestrian dataset in csv format

clean up dateString			to make the month two digits as 0 padded  i.e.   m/dd/yy  should be mm/dd/yy e.g.    date as 9/20/14   should become    09/20/14
	find
\r\n([0-9])/
	replace with
\r\n0\1/


now for the middle dd  of   mm/dd/yy    i.e.  mm/d/yy  should be mm/dd/yy   e.g.  05/9/15   should become  05/09/15
	find
\r\n([0-9][0-9])/([0-9])/
	replace with
\r\n\1/0\2/

now for the year part   i.e.  mm/dd/yy should become mm/dd/yyyy    e.g.  09/23/17  should become  09/23/2017
	find
\r\n([0-9][0-9])/([0-9][0-9])/([0-9][0-9])\r\n
	replace with
\r\n\1/\2/20\3\r\n

finding the missing value entries for date and count
e.g. 2007,05,1,a,10/05/2007,     has count 7503
^\d{4},\d{2},\d,[a-z],\d{2}/\d{2}/\d{4},

e.g. 2007,05,1,a,-,              has count 21
^\d{4},\d{2},\d,[a-z],-,
to check even further
^\d{4},\d{2},\d,[a-z],-,-,,,-,\d  which also has count 21

replaced the 21 entries with missing values for the count and the date. Currently the entries look like this
2008,05,0,m,-,-,,,-,102......
after replacement it should be like this
2008,05,0,m,,,,,,102.....

find
(^\d{4},\d{2},\d,[a-z]),-,-,,,-,(\d)
replace with
\1,,,,,,\2
